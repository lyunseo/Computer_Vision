# 포토샵 프로그램 만들기
> Open-cv를 이용한 간단한 포토샵 프로그램 구현

## 프로그램 기능 
  * 이미지 지정하여 열기
  * 다양한 기능 이미지에 적용가능
    > 반전, 흑백, 노이즈제거, 마스크, 회전, 확대축소, 히스토그램, 블러, 캐니엣지, 볼록렌즈, 관심 영역 지정, 얼굴 인식 


## 설치
```
pip install opencv-python
pip install pyside6
pip install matplotlib
pip install numpy
```
